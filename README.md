# Análise de Ações na Bolsa de Valores e Composição de Inadimplência de Emprestimos e seus Maiores Indicadores

A primeira parte desse projeto tem como intuito analisar ações do mercado financeiro, e a segunda parte tem como foco a composição do perfil de pessoas que fizeram empréstimos em bancos e se tornaram inadimplentes

## Objetivo

O objetivo da primeira parte desse projeto é analisar como foi o desempenho de 3 ações do Mercado Financeiro no período de 100 dias, já o da segunda parte, o foco maior é analisar os perfis inadimplentes de uma base de dados de empréstimos bancários

## Parte 1 - Ações Analisadas

- **PETR4.SA** - Petrobrás
- **ITUB4.SA** - Banco Itaú
- **VALE3.SA** - vale

## Parte 2 - Base de Empréstimos
- **Base de Dados:** Risco de Crédito (Kaggle)
- **Caminho:** /kaggle/input/credit-risk-dataset/credit_risk_dataset.csv

## Metodologia - Parte 1

1. **Extração dos Dados** utilizando a biblioteca python do yahoo.  
2. **Realizar Cálculos Financeiros** para obter o desempenho das ações.  
3. **Visualização dos resultados** por meio de gráficos de linha e boxplot utilizando o plotly.

## Metodologia - Parte 2

1. **Extração dos Dados** utilizando o site da kaggle.  
2. **Realizar Limpeza dos Dados**.
3. **Criar Análises** através da resposta de perguntas
4. **Visualização dos resultados** por meio de gráficos do Seaborn.

## Tecnologias Utilizadas

- **Python 3**
- **Pandas** - Manipulação e análise dos dados  
- **Yfinance** - Extração de dados financeiros do yahoo

- **Seaborn** - Criação de Gráficos
